# docs_repo — Documentación MCP Knowledge Hub

Fuente única de verdad para documentación del proyecto.

## Estructura
- `docs/` - Documentación general
- `bugs/` - Registro de bugs y resoluciones
- `_auto/` - Documentos generados automáticamente
- `flows/` - Diagramas y flujos
- `adr/` - Architecture Decision Records
- `company_projects/` - Proyectos de la compañía
- `staging/` - Documentos en revisión
- `inbox/` - Entrada de nuevos documentos
- `processed/` - Documentos procesados e indexados
