Arquitectura: MCP + OpenAI Embeddings + Qdrant

Elige una vista para ver el flujo.

Rol de cada componente

  • OpenAI API: Convierte texto en vectores (embeddings). Se usa al indexar cada chunk y al buscar (query → vector).
  • Qdrant: Guarda vectores y payload. Responde “puntos más parecidos a este vector”. No interpreta texto.
  • MCP Gateway: Orquesta: recibe búsqueda → embed query → consulta Qdrant → devuelve resultados.
