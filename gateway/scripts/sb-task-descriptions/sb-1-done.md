## Qué se hizo
Se definió la lista ClickUp para bugs y la convención BUG-Título en documentación.

## Código / archivos
- `gateway/docs/SUPERVISOR-BUGS.md`: lista (LIST_ID o auto-descubrimiento), convención de tareas con título `BUG-...`, y dónde se escribe la solución (descripción, sección "Solución sugerida").

## Cómo se usa
El supervisor usa la misma lista que el resto del proyecto; las tareas de bug se identifican porque el título empieza por `BUG-`.
